### Final Model 微调配置

在各变体 100 epoch 基础权重上，追加 2 epoch FLIR 微调，激活正则化损失：

```

tv_loss_weight  = 0.05    # L_tv = TotalVariation(pseudo_rgb)
logit_reg_weight= 0.02    # L_logit = sigmoid(logits).mean()
pseudo_clamp    = 0        # 不限制像素范围（保持基础权重兼容性）
residual_scale  = 1.0      # 残差缩放保持原行为
```

## 四、实验分析

### 4.1 消融维度 B：原型策略的重要性

**B-2（冻结均值原型）vs C-3（动态 GMM 单类别原型）**：

- FLIR：C-3 基础 +0.071，微调后 +0.056 → 动态原型持续有益
- M3FD：C-3 基础 +0.048，微调后 +0.030 → 跨域泛化也更好
- LLVIP：两者相近（-0.019），行人单类任务差异较小

**结论**：动态 GMM 原型（RPCA 核心机制）比静态均值原型在多数场景下有明显优势，尤其是多类别场景（FLIR/M3FD）。静态均值原型在纯行人场景（LLVIP）劣势不明显，说明人体的光谱特征相对稳定。

### 4.2 消融维度 C：CMSS 策略对比

**C-1（随机）vs C-2（GMM 多类）vs C-3（GMM 单类）**：

微调后三数据集均值：C-1 (0.534) > C-3 (0.489) > C-2 (0.497)

- **C-1 在微调后异常强劲**，尤其 LLVIP +0.1159 增益最大。这可能是因为随机掩码导致模型对"任意遮蔽"的鲁棒性更强，而 Final Model 的正则化损失（L_id/L_tv/L_logit）对随机掩码训出的特征有更好的正则化效果。
- **C-3 在微调后增益最小**（Δ均值仅 +0.013），说明 GMM 单类别的特征表示已经较为成熟，正则化收益边际递减。
- **M3FD 上 C-3 微调后反而下降 -0.014**，可能是 M3FD 训练域与 FLIR 不一致，2 epoch FLIR 微调对跨域 M3FD 稍有负迁移。

### 4.3 Final Model 微调的整体评估

| 维度 | 评价 |
|------|------|
| FLIR 域内 | 全部变体提升，C-1 +4.4% 最显著 |
| M3FD 跨域 | 小幅提升（+0.4%～+0.9%），C-3 轻微下降 |
| LLVIP 跨域 | C-1/C-2 大幅提升（+11%～+13%），B-2/C-3 中等提升 |
| 整体 | 正则化损失对 LLVIP 行人检测效果显著，对 M3FD 多类效果有限 |

**推测原因**：L_id（MSE 约束 pseudo_rgb 接近 IR）和 L_tv（空间平滑）对行人这类形状规则的目标效果好；M3FD 包含车辆、路灯等频繁变化的目标，纹理约束作用有限。

### 4.4 最终推荐配置

综合三数据集微调后均值：

```
最优变体：C-1（GMM 退化对照）+ Final Model 微调
  FLIR  mAP@0.5 = 0.5981
  M3FD  mAP@0.5 = 0.1926
  LLVIP mAP@0.5 = 0.8101
  三数据集均值   = 0.5336
```

> **注意**：C-1 的强表现是一个值得关注的 negative result——随机掩码 + 正则化组合的泛化性超过了精心设计的 GMM 策略。这可能意味着 CMSS 的价值更多在于"提供训练多样性"而非"精确的语义感知掩码"。建议后续实验验证是否因为 C-1 的 dropout 效应导致了更好的特征鲁棒性。
